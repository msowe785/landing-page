# Sound & Solitude — Responsive Music Landing Page

A beautifully designed, fully responsive landing page featuring a curated collection of music playlists. This project was built as part of **The Odin Project** curriculum to master HTML, CSS Flexbox, and responsive web design.

## 🚀 Features
- **Custom Branding:** Dark-themed aesthetic tailored for a late-night music vibe.
- **Fully Responsive:** Adapts flawlessly to desktop, tablet, and mobile screens.
- **Semantic HTML:** Structured with modern, accessible HTML5 tags.
- **Pure CSS Flexbox:** Built entirely with Flexbox without relying on external CSS frameworks.

## 🛠️ Built With
- **HTML5** - For the structure and skeleton of the site.
- **CSS3** - Custom styling, Flexbox layouts, and media queries.
- **Google Fonts** - Featuring the geometric and modern **Poppins** font family.
- **Flaticon** - Used for the header brand asset.

## 💡 What I Learned
During this project, I pushed my frontend skills to the next level by:
- **Going Beyond the Prompt:** While media queries weren't required for this Odin Project assignment, I chose to implement them on my own to ensure a seamless mobile-friendly experience.
- Deepening my understanding of the CSS Box Model and Flexbox alignment.
- Organizing project files and assets in a clean and simple structure for easy navigation.
